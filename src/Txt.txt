## react intro

    What is React?

    1) React is a JavaScript library (not a framework) developed by Facebook (Meta) for building user interfaces (UIs), especially Single Page Applications (SPAs).

    2) It helps developers build reusable UI components that update efficiently when data changes.

    Instead of reloading a whole webpage when something changes, React updates only the parts of the UI that need to change.

    -------
    Why do we need React?

	It makes UI development faster, cleaner, and more efficient.

	Provides a better user experience with smooth updates without page reloads.

## Routing in react and nested routing..

    Routing means navigation between pages in a web application without refreshing the browser.
    In traditional websites,clicking a link relads the whole page.
    In react (SAP) ,Routing lets us changes the view  without reloading.

        better performance (no full page reloag)
        smooth user experiance
        Control over navigation without serrvar request
    
    commend: npm install react-router-dom

    structure:
            <BrowserRoute>
            <Routers>
            <route path={} element={} />
            </Routers>
            </BrowserRoute>

    <BrowserRoute>--
        it tell react to use routing in your app.
        it keep your UI in syn with URL in the browser

    Routers---

        it's a container that holds all your different pages routes.
        it decides which page to show based on the URL

    route--

        it defines which component to show for a specific path (URL)
